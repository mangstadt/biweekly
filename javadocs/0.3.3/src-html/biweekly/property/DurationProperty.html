033/**
034 * <p>
035 * Defines a duration of time (for example, "2 hours and 30 minutes"). This
036 * property has different meanings depending on the component it belongs to:
037 * <ul>
038 * <li>{@link VEvent} - The duration of the event (used in place of a
039 * {@link DateEnd} property).</li>
040 * <li>{@link VTodo} - The duration of the to-do task (used in place of a
041 * {@link DateEnd} property).</li>
042 * <li>{@link VAlarm} - The pause between alarm repetitions.</li>
043 * </ul>
044 * </p>
045 * 
046 * <p>
047 * <b>Code sample:</b>
048 * 
049 * <pre class="brush:java">
050 * VEvent event = new VEvent();
051 * 
052 * Duration duration = Duration.builder().hours(2).minutes(30).build();
053 * DurationProperty prop = new DurationProperty(duration);
054 * event.setDuration(prop);
055 * </pre>
056 * 
057 * </p>
058 * @author Michael Angstadt
059 * @see <a href="http://tools.ietf.org/html/rfc5545#page-99">RFC 5545 p.99</a>
060 */
061public class DurationProperty extends ValuedProperty<Duration> {
062        /**
063         * Creates a duration property.
064         * @param duration the duration value (e.g. "2 hours nad 30 minutes")
065         */
066        public DurationProperty(Duration duration) {
067                super(duration);
068        }
069}
